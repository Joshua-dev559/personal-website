# Personal Website

This is a personal website built to showcase my skills and projects as a web developer. The website includes three pages:

1. **Home Page (`index.html`)**: Introduction, About Me, and Contact information.
2. **Skills Page (`skills.html`)**: Highlights my technical and soft skills.
3. **Projects Page (`projects.html`)**: Displays my projects, including the calculator project.

## Features

- Multi-section website using HTML
- Responsive design with CSS
- Navigation menu to move between pages

## Technologies Used

- HTML5
- CSS3

## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Joshua-dev559/personal-website.git
2. Open `index.html` in your web browser.
3. Navigate between pages using the top menu.
